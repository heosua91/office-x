## Overview

Screen Name: Meeting Room Status
URI: /office/schedule/rooms
Screen Overview: Global view of all organization meeting rooms organized as a Gantt Chart/Timeline. Users can instantly check availability across floors and navigate directly to create a booking or view specifics.

## Processing Details

```plain
Initial display
  • Initialize Date Scope to current day.
  • Call API: `GET /office/schedule/gantt?date=today`.
    ○ Response outlines all room profiles, mapped with an array of reservations (start time, end time, type, status, permission).
  • Render the timeline library module mapping rooms vertically.

Interaction
  • Data fetching checks user's authorization to view details. Masked attributes override response logic so the UI simply renders grey "Busy" boxes for private entries.
  • Users can hover over normal blocks for a tooltip detailing start/end time and Host. 
  • Click actions to navigate to meeting details will check if meeting is Internal (OFX-009) or External (OFX-004) based on the record type.
```
