## Actions

| 1 | New Registration Button | Press | Opens empty Word Registration Modal. |
| 2 | Word Edit Button | Press | Opens Word Registration Modal populated with existing item data. |

| 4 | Delete Button | Press | Destroys the word mapping in DB. |

## Processing Details

```plain
Initial display
  • Initialize using `GET /office/settings/dictionary` pulling an array of registered words.
  • Populate the Word List table.

Interaction
  • User types in "Search Box": Natively filters the Word List table dynamically by text matching Word or Furigana arrays.

Action
  • "(M) Save Button" button is pressed:
    ○ Validates if (M) Word exists.
    ○ Call API: `POST` or `PATCH` on `/office/settings/dictionary` item.
    ○ If successful: Close modal. Reload Word List.
    ○ If failed: Show Error Message ID.
  • "Delete Button" confirmed:
    ○ Call API: `DELETE /office/settings/dictionary/:termId`.
    ○ If successful: Remove item from List.
```
