## Overview

Screen Name: Upcoming Meetings List
URI: /office/schedule/upcoming
Screen Overview: Aggregated view of all upcoming scheduled meetings across the organization (incorporating both client and internal meetings). Access lists are filtered according to user sharing and privacy settings.

## Processing Details

```plain
Initial display
  • Fetch default upcoming list via `GET /office/meetings?timeframe=upcoming`.
  • Renders dataset in chronological order. Data is pre-filtered by the backend returning masked responses for records set to strict privacy (visible only to participants and host).

Interaction
  • Data interaction handles search keyword entries re-fetching the list dynamically.
  • Clicking a row routes user depending on the metadata type of the record. Record types distinguishing internal vs client mappings dictate the target UI path to load.
```
