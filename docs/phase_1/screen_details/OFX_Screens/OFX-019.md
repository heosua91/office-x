## Overview

Screen Name: Privacy & Defaults
URI: /office/settings/privacy
Screen Overview: Configure default sharing behaviors for user's reservations. E.g., setting whether new schedules are completely transparent (titles visible to all staff) or strictly private (showing only "Busy") unless specifically toggled per meeting.
