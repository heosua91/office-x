## Overview

Screen Name: Privacy & Security Settings
URI: /office/settings/privacy
Screen Overview: Configure default sharing behaviors for user's reservations. Settings for Public Default of meeting minutes (Only Me / Only Participants) and configuration of display separation for "Internal Memo" and "Share for Clients".

## Processing Details

```plain
Initial display
  • Initialize component states matching user's stored preferences via API response.
  • Disable "Save Button" initially.

Interaction
  • Enable "Save Button" if any dropdown or toggle is modified by the user (dirty checking).

Action
  • "Save Button" button is pressed:
    ○ Call API: `PATCH /office/settings/privacy` with payload of configured items.
    ○ If successful: Show Success Message ID and navigate back.
    ○ If failed: Show Error Message ID.
  • "Cancel Button" button is pressed:
    ○ Discard changes and return.
```
