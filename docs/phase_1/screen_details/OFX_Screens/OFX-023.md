## Overview

Screen Name: Internal Meeting Details
URI: /office/meetings/internal/:id/ai
Screen Overview: Displays details for finished internal meetings (Transcripts, AI Minutes, ToDo List). Supports audio playback linked to transcripts. Allows launching "AI Chat" referencing meeting minutes.

## Processing Details

```plain
Initial display
  • Call API: `GET /office/meetings/:id/ai` or the internal equivalent.
  • Render Meeting Basic Info, Minutes, ToDos, Transcripts, and Attached Files.
  • Determine user privileges to show or hide "Edit Button".

Interaction
  • User switches Tabs: Conditional rendering swaps the active viewer layout without remounting audio player.
  • User clicks "AI Chat Invoke Button": Mounts chatbot component floating above UI.
  • User clicks "Todo Status": Update UI to crossed out instantly (optimistic update).

Action
  • "Todo Status" is toggled:
    ○ Call API: `PATCH /office/action-items/:id` or similar Todo update endpoint.
  • "Edit Button" button is pressed:
    ○ Navigate to OFX-024.
```
