## Overview

Screen Name: Internal Meetings List
URI: /office/meetings/internal
Screen Overview: Dedicated view for internal meetings managed via folders based on topic or department. Displays past history logs (minutes, ToDo) and upcoming reservations aligned to chronological order in a Timeline view.

## Processing Details

```plain
Initial display
  • Call API to reconstruct user's folder hierarchy structure.
  • Auto-select the "General" or first index folder node default.
  • Call API `GET /office/meetings?type=internal&folder_id=default` to pull chronological points matching both historical meeting abstracts (completed sessions) and ongoing bookings.

Interaction
  • When a new folder is selected on the sidebar, clear state and pull timeline representing context of the selected entity.
  • Folders can be natively managed (Right-click logic or explicit icons) to trigger create/rename/delete endpoints. Deleting a folder mandates shifting loose schedules to root folder before termination to avoid orphan data.
  • Clicking an item on the timeline navigates to OFX-009 if is future schedule or OFX-023 if it belongs to history data.
```
