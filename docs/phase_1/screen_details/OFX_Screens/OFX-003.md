## Overview

Screen Name: Booking Adjustment & Smart URL Generation
URI: /office/meetings/new-client
Screen Overview: Form to create a smart booking link for a client meeting. The user selects a target client, meeting duration, format, and participants. The system generates a dedicated URL that guests can use to pick a slot that satisfies the host/team availability and room availability (via AND logic).

## Processing Details

```plain
Initial display
  • Fetch AI Templates list to populate the dropdown.
  • Fetch company users for Participant multi-select (default self selected).

Interaction
  • Form captures core settings for the meeting URL.
  • The Company Name combobox searches existing companies. If the user types a new string, it will be treated as a new company registration payload upon DB insertion.

Action
  • Generate URL button pressed:
    ○ Validates inputs.
    ○ Sends payload to `POST /office/meetings/smart-url`.
    ○ The backend will register the company into master if it's new.
    ○ The backend returns a unique `invite_token_url` (which maps to the guest's GRES-001 screen).
    ○ Show a success modal confirming the link with a "Copy Link" button.
```
