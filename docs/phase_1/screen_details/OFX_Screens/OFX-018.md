## Processing Details

```plain
Initial display
  • Call API: `GET /office/settings/integrations` to fetch the current linked calendar status.
  • If linked: Display account name, show "Unlink Button" and "Calendar Sync Scope".
  • If unlinked: Show "Authentication Request Button", hide "Unlink Button".

Interaction
  • User selects Google or Outlook: Sets state for OAuth redirection.
  • User clicks "Unlink Button": Show a confirmation modal to avoid accidental unlinking.

Action
  • "Authentication Request Button" is pressed:
    ○ Triggers OAuth redirect to the selected provider.
  • "Unlink Button" confirmed:
    ○ Call API: `DELETE /office/settings/integrations/calendar`
    ○ If successful: Refresh layout to Unlinked state.
    ○ If failed: Show Error Message ID.
  • "Save Button" button is pressed:
    ○ Call API: `POST /office/settings/integrations` to save sync preferences.
    ○ If successful: Show Success Message and return.
    ○ If failed: Show Error Message ID.
```
