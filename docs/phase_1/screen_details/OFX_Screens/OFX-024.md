## Overview

Screen Overview: Screen for manually editing internal meeting minutes, Todo lists, and internal memos. Allows AI regeneration with different templates.

## Processing Details

```plain
Initial display
  • Initialize using `GET /office/meetings/:id` and subresources.
  • Determine via AI Usage API if quotas are reached. If yes, disable "AI Regen Button" and show "AI Quota Warning".

Interaction
  • Form transitions to Dirty when text blocks, drop downs, or tables are interacted with.
  • If "Sharing Settings" selected is "Designated Users", show the corresponding Combo Box.

Action
  • "AI Regen Button" button is pressed:
    ○ Modal warns of overwriting current text.
    ○ If accepted: Call API: `POST /office/meetings/:id/regenerate` with `templateId`. Updates local text blocks.
  • "Delete Button" button is pressed:
    ○ Call API: `DELETE /office/meetings/:id`.
    ○ If successful: Redir to Dashboard or Meeting List.
  • "Save Button" button is pressed:
    ○ Call API: `PATCH /office/meetings/:id/summary` incorporating minutes, memos, todos.
    ○ If successful: Diff analysis may show Dictionary Popup. If shown, handle user response. Finally redirect back to OFX-023.
    ○ If failed: Show Error Message ID.
```
