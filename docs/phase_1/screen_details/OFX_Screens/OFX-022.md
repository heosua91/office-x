## Overview

Screen Name: Google Drive Integration
URI: /office/settings/integrations/drive
Screen Overview: Uses OAuth 2.0 to setup a read-only Google Drive access token, allowing users to reference Google Drive materials natively during scheduling processes.

## Processing Details

```plain
Initial display
  • Initialize using configuration variables to determine integration state.
  • Check if connected: if connected show the email (using Integration Status text), the "Re-auth/Change Account" and "Unlink Button" buttons.
  • Hide "OAuth Link Button". If unlinked, do the reverse.

Interaction
  • None strictly specific to form fields.

Action
  • "OAuth Link Button" or "Re-auth/Change Account" button is pressed:
    ○ Triggers redirect loop out to external endpoint. Returns with Authorization Code matching state internally.
  • "Unlink Button" confirmed:
    ○ Delete association on the server for Google integration.
    ○ If successful: Reload screen state to Unlinked.
    ○ If failed: Display Error Message ID.
```
