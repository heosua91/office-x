## Processing Details

```plain
Initial display
  • Call `GET /office/dashboard` to fetch widget data:
    ○ Today's Schedule: Syncs with Calendar API + internal reservation DB. Displays chronologically.
    ○ Room Status: Aggregates real-time state of registered `meeting_rooms`.
    ○ AI Usage: Aggregates `usage_logs` for current month user.
  • Render AI Alert icon if remaining proportion <= 10%.

Interaction
  • Schedule list is read-only but interactive. Clicking a schedule item branches navigation:
    ○ Client Meeting (Upcoming/Ongoing) → OFX-004
    ○ Internal Meeting (Upcoming/Ongoing) → OFX-009
    ○ Internal Meeting (Ended) → OFX-023
    ○ Client Meeting (Ended) → OFX-013
```
