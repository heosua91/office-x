## Processing Details

```plain
Initial display
  • Fetch integrated app configurations utilizing `GET /office/settings/integrations`.
  • Populate the Table items matching saved configs. Check if browser push notifications currently hold granted permissions, switch toggle accordingly.

Interaction
  • Typing in URL fields or toggling notifications marks form as dirty and enables the Save button.
  • If Push Notification toggle turned ON, invoke browser API `Notification.requestPermission()`. If denied, revert toggle to OFF and prompt user to allow notifications from browser settings.

Action
  • "Save Button" button is pressed:
    ○ Emits `POST /office/settings/integrations` with payloads containing the Webhooks and status configurations.
    ○ If successful: Show Success Message ID.
    ○ If failed: Show Error Message ID.
```
