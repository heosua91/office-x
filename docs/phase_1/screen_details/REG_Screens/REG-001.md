## Processing Details

### 初期表示/Initial display
- Display each item according to the screen item definitions.
- Set the "Next" (ID: 5) button to a disabled state.

### 画面更新時/Interaction
- When the "Agree" (ID: 4) checkbox is toggled:
    - If the "Agree" checkbox is checked:
        - Set the "Next" (ID: 5) button to an enabled state.
    - Otherwise:
        - Set the "Next" (ID: 5) button to a disabled state.

### アクション発生時/Action
- **Màn hình vừa load/Immediately after screen load**:
    - Execute API request:
        - Endpoint: `/auth/register/terms`
        - Method: `GET`
    - **If successful**:
        - Populate "Terms of Use Text" (ID: 2) and "Privacy Policy Text" (ID: 3) with the retrieved content.
    - **If failed**:
        - Display an error message (system error) and potentially provide a "Retry" option.

- **Nút "Next" (ID: 5) được nhấn/ "Next" button is pressed**:
    - Perform a validation check according to the "Validations" section.
    - If there are no errors:
        - Transition to the next screen (REG-002).

- None.
