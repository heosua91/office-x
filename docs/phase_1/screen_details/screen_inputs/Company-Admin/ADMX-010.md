## データ登録方式一覧

| マスタ種類 | 登録方式 | 管理画面での操作 |
|---|---|---|
| 業者名 | 初期デフォルト値として事前設定 | 追加・編集・削除・受付拒否設定 可能 |
| 用件 | 初期デフォルト値として事前設定 | 追加・編集・削除・受付拒否設定 可能 |
| 部署名 | ユーザー一覧CSVインポート時に自動登録（ADMX-003） | 追加・編集・削除 可能 |
| フロア | 会議室一覧CSVインポート時に自動登録（ADMX-007） | 追加・編集・削除 可能 |
| 会議室備品 | 会議室一覧CSVインポート時に自動登録（ADMX-007） | 追加・編集・削除 可能 |
