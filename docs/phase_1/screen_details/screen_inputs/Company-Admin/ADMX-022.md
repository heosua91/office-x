ADMX-022

プラン変更・決済 (Stripe更新 API連携)

企業管理者がプラン（利用枠・機能権限のアップグレードやダウングレード等）を変更する際に、Stripe Checkoutへと向かうための中継・ゲートウェイ画面。
※前段（システム全体）の制御として、特別プラン契約中のユーザーが当画面に直接アクセスした場合は403エラー等でブロックするか、ADMX-017へリダイレクトする前提とする。
