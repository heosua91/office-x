AUTH-008			

初回パスワード強制再設定			

TNG代理作成等で発行された初期パスワードを利用して初回ログインした場合に表示されます。セキュリティのため、強制的に新しいパスワードの再設定を求めます。設定完了後、正規のログインセッションを発行します。
