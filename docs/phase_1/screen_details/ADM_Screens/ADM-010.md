## Overview

Screen Name: Overview Analytics AI Usage
URI: /admin/analytics
Screen Overview: Display AI summary usage status of all contract companies, prepaid/postpaid ratios, and a list of companies that have exceeded their quota. Used by TNG Admins to monitor AI resource consumption.

## Processing Details

```plain
Initial display
  • Initialize Period Filter to the current month by default.
  • Render all statistic cards and empty tables before API response.

Interaction
  • User selects Date from Period Filter:
    ○ Triggers a data fetch for the new month, updating the charts and statistics.

Action
  • Screen Load / Period modified:
    ○ Call API: GET /tng/analytics/usage (with period parameters)
    ○ If successful: 
        - Update Total AI Usage, Postpaid Using Companies, Postpaid Total.
        - Draw Usage Chart (Pie chart).
        - Populate Over-quota Company Table.
    ○ If failed: Display Error Message ID.
  • Details Button is pressed:
    ○ Store current UI state if needed, navigate to ADM-002 passing the company identifier.
  • Export Button is pressed:
    ○ Trigger data download through API or frontend generation. Prompt CSV save dialog.
```
