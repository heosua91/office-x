## Overview

Screen Name: AI Overage Billing Management
URI: /admin/billing/overage
Screen Overview: Aggregation of AI overage usage (postpaid) per company, and confirmation of billing amounts. Used by TNG Admins to oversee billing totals and check usage details.

## Processing Details

```plain
Initial display
  • Initialize Billing Month to current month.
  • Initialize empty Company Name Search.
  • Fetch Monthly Trend Chart metrics and render chart.

Interaction
  • Update Filters -> Wait for Search Button to trigger API refresh.

Action
  • Screen Load:
    ○ Triggers default search/filter logic based on initial date mappings.
    ○ Fetch Chart data.
  • Search Button is pressed:
    ○ Read parameters (Month, Company).
    ○ Call API: GET /tng/analytics/usage (with postpaid filters) or /tng/billing/invoices logic.
    ○ If successful: Render Billing List Table metrics.
    ○ If failed: Display Error Message ID.
  • Export Button is pressed:
    ○ Generates CSV from active table dataset. Prompt user download.
```
