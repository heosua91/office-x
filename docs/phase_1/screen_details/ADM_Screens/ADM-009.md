## Overview

Screen Name: Equipment Master Management Screen
URI: /admin/equipment-catalog
Screen Overview: Add/edit/change prices for rental items (tablets, stands, microphones, etc.) and sales items. Capable of setting rules (keep/update across the board) for price application to existing contracts.

## Processing Details

```plain
Initial display
  • Initialize Equipment List Table.
  • Disabled Delete buttons for any items tied to an existing tenancy contract.

Interaction
  • Inside Modal:
    ○ Choose "Rental" -> Display "Monthly Fee". Hide/Clear "Sale Price".
    ○ Choose "Sale" -> Display "Sale Price". Hide/Clear "Monthly Fee".
    ○ In Edit Modal, if the Price value is changed -> Display "Pricing Revision Rule" radio options.
    ○ If "Pricing Revision Rule" = "Update across board" -> Display "Application Start Date".

Action
  • Screen Load:
    ○ Call API: GET /tng/catalog/devices
    ○ If successful: Populate Equipment List Table.
    ○ If failed: Display Error Message ID.
  • Add/Update Button is pressed:
    ○ Evaluate front-end validation limits.
    ○ Call API: POST /tng/catalog/devices (Creation logic or Update logic with parameters).
    ○ If successful: Show Success Message (S001), reload table, close Modal.
    ○ If failed: Display Error Message.
```
