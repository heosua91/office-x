## Overview

Screen Name: Terms & Privacy Policy Management
URI: /admin/terms
Screen Overview: Interface for TNG Admins to update and version control the Terms of Use and Privacy Policy documents without requiring engineering support. Allows previewing and publishing Markdown/HTML content, with mandatory dates for rollouts.

## Processing Details

```plain
Initial display
  • Publish parameters empty.
  • Preview Button disabled. Register Button disabled.
  • Published Version List fetches previous versions and displays.

Interaction
  • User selects file via Select File Button:
    ○ Validate extension format locally. If pass, enable Preview Button.
  • Enable "Register / Publish" button once all marked 'Required' fields are populated properly.

Action
  • Screen Load:
    ○ Fetches previous version history list.
  • Register / Publish Button is pressed:
    ○ Post Multipart Form data including the file stream and JSON parameters.
    ○ If successful: Show Success Message (S001). Clear form. Pre-pend history list with new version details.
    ○ If failed: Display Error Message ID.
```
