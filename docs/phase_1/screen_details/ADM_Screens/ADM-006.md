## Overview

Screen Name: Master Plan Management Screen
URI: /admin/plans
Screen Overview: Manage the basic pricing plans (4 types) of OfficeX and the feature permissions available for each plan. Includes navigation to edit specific plan details.

## Processing Details

```plain
Initial display
  • Initialize Plan List Table.
  • All plan records are displayed.

Interaction
  • No dynamic interaction aside from navigating.

Action
  • Screen Load:
    ○ Call API: GET /tng/plans
    ○ If successful: Populate Plan List Table (Items 2-1 to 2-4).
    ○ If failed: Display Error Message ID (e.g. E001).
  • Plan Name Link or Action Button is pressed:
    ○ Navigate to ADM-007, passing the selected plan's identifier.
```
