## Overview

Screen Name: New Contract Company Registration (Proxy Creation)
URI: /admin/proxy/companies/new
Screen Overview: Admin proxy creation to skip standard Stripe/onboarding routines. Automatically sets first-use configurations and enforces a password reset flag for the initial administrator user when they first login.

## Processing Details

```plain
Initial display
  • Load available Dropdown items for 'Applied Plan'.
  • Set Initial Password textbox to random default 8-char secure generator hint.
  • Check "Send ID Auto-Email" by default.
  • Registration Action is disabled.

Interaction
  • Typing satisfies "Required" thresholds, Registration button enabled.

Action
  • Register Execution Button is pressed:
    ○ Client validations against email string shapes and password rules.
    ○ Calls DB creation / proxy sign-up API without Stripe dependency rules.
    ○ If successful: Show Success message (S001), navigate back to ADM-001 list.
    ○ If failed: Display Error Message ID based on uniqueness faults.
  • Back Button is pressed:
    ○ Dump State, navigate to ADM-001.
```
