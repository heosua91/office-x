## Overview

Screen Name: Contract Company List
URI: /admin/contracts
Screen Overview: Displays a list of registered contract companies. Information includes company name, contract status, contract plan, contract expiration date, number of registered users remaining, AI summarize time remaining, and payment status. Capable of searching and filtering.

## Processing Details

```plain
Initial display
  • Initialize search keyword to empty.
  • Initialize contract status to "All".
  • Initialize contract plan to "All" (Load available plans from master).
  • All buttons and inputs are enabled.
  • Fetch company list with default pagination.

Interaction
  • No dynamic real-time interaction logic aside from selecting fields.

Action
  • Screen Load:
    ○ Call API: GET /tng/companies
    ○ If successful: Populate Company List Table (Items 9-1 to 9-7). Paint status flags / thresholds accordingly (e.g. 30 days expiry to red, 10% AI remaining to yellow).
    ○ If failed: Display Error Message ID (e.g., E001).
  • Search button is pressed:
    ○ Validate input criteria constraints (Max length).
    ○ Call API: GET /tng/companies with search parameters.
    ○ If successful: Update the Company List Table with search results.
    ○ If failed: Display Error Message ID.
  • Clear button is pressed:
    ○ Reset all search controls (Keyword, Status, Plan) to their default states.
    ○ Trigger Search button logic automatically.
  • CSV Export button is pressed:
    ○ Call API to generate CSV data matching the current filter constraints.
    ○ If successful: Prompt browser download.
    ○ If failed: Display Error Message ID.
```
